💻 Laptop Price Predictor
This is a Streamlit web application that predicts laptop prices based on user-specified features. It utilizes machine learning models to provide accurate price estimations. 🚀

✨ Features
🔹 User-Friendly Interface: Easy-to-use Streamlit interface for inputting laptop features. 🖥️
🔹 Multiple Models: Supports Linear Regression, XGBoost, and LightGBM models. 🧠📊
🔹 Currency Conversion: Price predictions can be displayed in USD, EUR, or TRY. 💵 💶 ₺
🔹 Random Image Display: Displays a random laptop image with the prediction. 🖼️
🔹 Responsive Design: Adapts to different screen sizes. 📱💻

🛠️ How to Use
1️⃣ Select Laptop Features: Use the sidebar to input the laptop's screen size, RAM, storage, PPI, weight, brand, processor, GPU, and operating system. ⚙️
2️⃣ Choose Currency: Select the desired currency (USD, EUR, or TRY). 💲
3️⃣ Select Model: Choose the machine learning model for prediction (Linear Regression, XGBoost, or LightGBM). 🤖
4️⃣ Predict Price: Click the "Predict Pc Price" button to get the price prediction. 🎯
5️⃣ View Results: The predicted price will be displayed in the selected currency, along with a random laptop image. ✅
